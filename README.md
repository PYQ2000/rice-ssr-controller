This repository provides the control and acquisition code for a rice seed-setting-rate (SSR) measurement device.
The software coordinates conveyor belts, airflow separation, precision weighing, and camera triggering to support automated and repeatable rice phenotyping experiments.

## New UI Features

- Real-time dual camera preview shown under the recording controls.
- Preview continues during recording by reusing frames from the recorder.
- Preview can be disabled to reduce CPU usage.
- Preview scale slider (default 20%) to save space while monitoring.
